Sizing Unit

  1. px : Pixels
  2. em : units for the font-size property will be relative to the font-size of the parent element.
  3. rem : units sizes will always be relative to the font-size of the root html element.
  4. vmin : vmin stands for viewport minimum. The vmin function is used to set the size of an element as a percentage of the minimum value between the viewport width or height.
  5. vmax :vmax stands for viewport maximum. The vmax function ranks elements as a percentage of the maximum value between the viewport width or height.
